# Roadmap

This roadmap focuses on making codex-dashboard easier to trust, easier to run, and safer to share.

## Near Term

- Improve privacy mode coverage for project names, session names, and screenshots.
- Add setup screenshots for macOS and Windows release packages.
- Add a release validation checklist to ensure bundled files use sample data only.
- Improve sample data so first-time users can understand each dashboard panel without local logs.

## Maintenance

- Keep pricing assumptions easy to review and update.
- Keep local-only generated files ignored by default.
- Preserve a simple install path for non-technical users.
- Expand tests for the Codex session log parser.

## Non-Goals

- Hosted analytics service.
- Uploading local Codex session logs.
- Account-level tracking or remote telemetry.
